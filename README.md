## Description
Ce projet permet de détecter automatiquement les biais linguistiques présents dans les réponses d’un chatbot de service client
(ton rude, accusation, manque de politesse, généralisation, langage non inclusif)
et de proposer des reformulations plus neutres et inclusives.

---

## Fonctionnalités
- Détection des biais linguistiques
- Classification par type de biais
- Reformulation automatique des réponses
- Statistiques globales
- Application interactive avec Streamlit

---

## Technologies
- Python  
- Pandas  
- Regex  
- Streamlit  

---

## Lancement
